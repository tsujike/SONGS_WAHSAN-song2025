# トキエア合宿テーマソング

## 概要

このディレクトリには、**HIU冬合宿 Island Hopping x トキエア**のテーマソング作成に関する資料が含まれています。

## ファイル構成

- **`collected_phrases.md`** - PR_SNS-Analyticsディレクトリから収集した、テーマソングに使えそうなフレーズやテキストのまとめ
- **`SUNO_PROMPT.md`** - SUNO AI音楽生成ツール用のプロンプト集
- **`README.md`** - このファイル

## 合宿情報

### 正式タイトル
**「HIU冬合宿 Island Hopping x トキエア」**

### 正式コピー
**『ぶっ飛ぶ！ by TAKAFUMI HORIE INNOVATION UNIVERSITY』**

### 日程
- **2026年2月3日(火) 〜 2月5日(木)**（2泊3日）
- **ルート**: 神戸空港 → 隠岐島 → 沖永良部島 → 神戸空港

## テーマソングのコンセプト

### メインメッセージ
- 「自由な空の旅を手に入れる、常識を超える体験を」
- 「さぁ、最高の景色をみんなで見に行こうぜ！」
- 「ぶっ飛ぶ！」

### 音楽的な方向性
- 冒険的でエネルギッシュ
- みんなで歌えるキャッチーなメロディ
- 自由と挑戦の精神を表現
- 仲間との絆を感じられる内容

## 使用方法

1. **`collected_phrases.md`** を参照して、使いたいフレーズをーズを確認
2. **`SUNO_PROMPT.md`** から適切なプロンプトを選択
3. SUNO AIで音楽を生成
4. 必要に応じて歌詞やメロディを調整
