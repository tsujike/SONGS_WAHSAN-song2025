## 和ぁさんについて

和ぁさんは、徳島県美馬市小島駅周辺で営業しているセルフうどん店です。2023年1月にオープンした和ぁさん家は、うどん店に併設する形でパン屋と肉まん・餃子のお店が連なる複合ショップとして、「食のテーマパーク」を目指しています。

### 和ぁさん家のブランディング

**良いものを大切に。身体に直接取り入れるものだから「食」を和ぁさんは大切に考えています。**

読んでわかる原材料だけで作っている贅沢なパン。食べる楽しさと幸せ。心豊かな食生活を願い、良いものを丁寧にお客様にお届けすることを大切にしています。

### 和ぁさん家のわぁぱん

わぁぱんは、和ぁさん家®のパン屋です。「**おなかいっぱいの美味しい笑顔をお届けできますように**」。安心・安全な素材にこだわり、北海道産秋播小麦100%や国産生乳100%の牛乳・バターを使用した食パンや、常時30種類以上のコッペパンサンド（おかず系からスイーツ系まで）を販売しています。

#### わぁぱんのこだわり

- **厳選素材へのこだわり**：
  - **北海道産秋播小麦100%**：自然な香りと風味を大切に
  - **国産生乳100%のバター・クリーム・牛乳**：安心・安全な素材へのこだわり
- **その他の原材料**：アルゼンチン産はちみつ、天日塩、三温糖、フランス産ドライイースト、**卵不使用**
- **おいしさのヒミツ**：
  - 1.5斤以上の生地を1斤に焼き上げることで、独特のふわもち感が生まれます
  - 焼きたてのわぁぱんは耳までカリカリ。旨味の凝縮した耳は、やみつきになる美味しさです
  - 北海道産小麦と国産生乳100%のバター・クリームの自然な香り。そのままでも美味しいですが、焼くと香ばしく、家中が美味しい香りに満たされます
- **味わい**：きめ細かく、しっとりとした味わい。三温糖とアルゼンチン産はちみつの優しい甘さで、そのまま食べてもおいしい

#### 美味しく召し上がるために

- **焼きたてのわぁぱん**：そのままちぎってお召し上がりください。今までにない食感と風味をお楽しみいただけます
- **冷えた場合**：バター・生クリームをふんだんに使用しているため、冷えると少し固くなります。レンジで軽く温めるか、トーストしていただくとバターやはちみつの風味が際立ちます
- **保存方法**：食べきれない分はスライスしてから1枚ずつラップをして、冷凍保存をしてください。冷蔵したものは召し上がる直前にトーストしていただくと、さらに美味しく召し上がれます

## 1. 募集内容

「わぁぱん号」は、和ぁさん家の移動販売車です。 コンセプトは、「おなかいっぱいの美味しい笑顔をお届けできますように」。 安心して食べられる、家族に毎日食べてもらいたい味。家庭料理こそが一番のご馳走でありますように――そんな願いを歌に込めたいと思っています。

今回募集するのは、童謡のように口ずさみやすく、ロバのパン屋さんのように子どもたちが自然と歌いたくなるような、繰り返しのある楽しいメロディの「わぁぱん号の歌」です。 歌詞とメロディを合わせた「歌詞付きの曲」としてご応募ください。

さらに、曲調は 親しみのある、楽しい雰囲気で、住宅地でスピーカーから流したときにみんなの耳に残り、聞いた人が楽しくなるようなもの を目指してください。
